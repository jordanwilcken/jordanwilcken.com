The Birthday API is an API I wrote for keeping track of birthday information. It's not the first API that I've written, but it is the first that I've written all by myself. I wrote it in C# using Microsoft's ASP.NET Web API. The database used for storing the birthday info is SQLite.

The base url for the API is www.jordanwilcken.com/birthdayapi. If you try browse to this location you'll be redirected to the documentation for the API. The documentation was generated using apiddocjs.
